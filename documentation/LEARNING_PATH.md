## üí° **TIPS FOR LEARNING**

### **1. Start Simple**
- Start with basic viewing (dashboard, inventory)
- Then learn creating (requests, applications)
- Then learn managing (approving, editing)

### **2. Practice Regularly**
- Practice daily, kahit 15 minutes lang
- Try different scenarios
- Test different roles

### **3. Use Guides as Reference**
- Keep guides open while practicing
- Refer to diagrams when confused
- Check troubleshooting section if may error

### **4. Test Everything**
- Test all features
- Try different scenarios
- Create test data
- Delete test data after

### **5. Ask Questions**
- If confused, check guides first
- If may error, check troubleshooting
- If may missing feature, check documentation

## üö® **COMMON MISTAKES TO AVOID**

### **1. Wrong Role**
- ‚ùå Trying to approve as Branch Manager (should be Central Office Admin)
- ‚úÖ Check your role first before doing actions

### **2. Wrong Status**
- ‚ùå Trying to approve already approved request
- ‚úÖ Check status first before doing actions

### **3. Missing Data**
- ‚ùå Creating request without selecting supplier
- ‚úÖ Always fill up required fields

### **4. Not Confirming**
- ‚ùå Not confirming delivery
- ‚úÖ Always confirm after receiving

### **5. Not Checking**
- ‚ùå Not checking inventory before ordering
- ‚úÖ Always check current status first
